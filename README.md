## Projetos em destaque

### Sigma360 Farma

Plataforma SaaS para gestão operacional e analítica de farmácias, com módulos de sanitização de clientes, conexões, recuperação comercial e BI operacional.

### OADF

Framework próprio de desenvolvimento orquestrado por IA, com papéis definidos, GitHub Issues como fonte da verdade, evidências, QA, Sentinel e execução controlada.

### Sigma360 Request Hub

Central de solicitações e governança operacional para evolução de produtos Sigma360.

## Como trabalho

- GitHub Issues como fonte da verdade
- Execução controlada por escopo
- Evidências antes de aceite
- Separação clara entre PO, Orquestrador, Executor, QA e Sentinel
- Deploy seguro com validação Hostinger-first
- Arquitetura orientada a dados, operação e rastreabilidade
